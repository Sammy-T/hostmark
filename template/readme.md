# hostmark

A knowledge-base and note app supporting markdown text.

## Files

Editors and admins can create and edit markdown files which are readable by all members.

> [!NOTE]
> Subdirectories are supported when providing the file name.
>
> i.e. `some-directory/another-directory/some-file.md`

## Notes

All members can create and edit their own notes.

Notes can be configured with the following privacy levels:

|  |  |
| -- | -- |
| public | Anyone can read. |
| protected | Members can read. |
| private | The note owner can read. |
